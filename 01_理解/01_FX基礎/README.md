# FXの基礎知識

## FXとは何か

Foreign Exchange（外国為替証拠金取引）の略。
異なる通貨を売買して、その差額で利益を得る取引。

例: 1ドル=150円のとき「ドルを買い」→ 1ドル=152円になったら「ドルを売る」→ 2円の利益

## 市場の特徴

- **24時間取引可能**（月曜早朝〜土曜早朝）
- **世界最大の金融市場**（1日の取引高 約7.5兆ドル ≒ 1,100兆円）
- **主要参加者**: 中央銀行、商業銀行、ヘッジファンド、個人トレーダー
- **レバレッジ**が使える（少ない資金で大きな取引が可能。国内は法規制により最大25倍）

## 取引時間帯（日本時間）

| セッション | 時間 | 特徴 |
|-----------|------|------|
| 東京 | 9:00-15:00 | 値動き小さめ。USD/JPYが活発 |
| ロンドン | 16:00-25:00 | 最も取引量が多い。EUR関連が活発 |
| ニューヨーク | 22:00-翌7:00 | 重要指標発表が多い。ボラティリティ高 |
| ロンドン×NY重複 | 22:00-25:00 | **最も値動きが激しい時間帯** |

## 基本用語

### 通貨ペア
2つの通貨の組み合わせ。左が「基軸通貨」、右が「決済通貨」。

- **EUR/USD** = ユーロ/米ドル（世界で最も取引量が多い）
- **USD/JPY** = 米ドル/円（日本人に最も馴染みがある）
- **GBP/USD** = 英ポンド/米ドル（値動きが大きい）

### メジャーペア vs マイナーペア
- **メジャー**: USD絡み（EUR/USD, USD/JPY, GBP/USD等）→ スプレッドが小さい
- **マイナー**: USD以外同士（EUR/GBP等）
- **エキゾチック**: 新興国通貨絡み → スプレッドが大きく初心者には不向き

### ロング / ショート
- **ロング（買い）**: 価格上昇を見込んで買う
- **ショート（売り）**: 価格下落を見込んで売る
- FXは「売りから入れる」のが株との大きな違い

### pips（ピップス）
価格変動の最小単位。

- USD/JPY: 0.01円 = 1pip（150.00 → 150.50 = 50pips上昇）
- EUR/USD: 0.0001ドル = 1pip（1.0800 → 1.0850 = 50pips上昇）

### ロット（Lot）
取引量の単位。

| 種類 | 通貨量 | 1pip の損益（USD/JPY） |
|------|--------|----------------------|
| 1ロット（標準） | 100,000通貨 | 約1,000円 |
| 0.1ロット（ミニ） | 10,000通貨 | 約100円 |
| 0.01ロット（マイクロ） | 1,000通貨 | 約10円 |

### レバレッジ
証拠金の何倍の取引ができるかを示す。

- 例: 10万円の証拠金 × レバレッジ25倍 = 250万円分の取引が可能
- **レバレッジが高い = リスクも高い**
- 初心者は **3-5倍** で始めるのが鉄則

### スプレッド
買値（Ask）と売値（Bid）の差。これが実質的な取引コスト。

- USD/JPY: 0.2-0.3pips（安い）
- GBP/JPY: 0.6-1.0pips（やや高い）
- **スプレッドが狭い通貨ペアを選ぶことがコスト削減の基本**

### スワップポイント
通貨間の金利差から生じる損益。ポジションを翌日に持ち越すと発生。

- 高金利通貨を買い・低金利通貨を売り → スワップ受取
- 逆なら → スワップ支払
- 日をまたぐトレードでは無視できないコスト/収益

## 注文の種類

### 成行注文（Market Order）
現在の市場価格で即座に売買する。

### 指値注文（Limit Order）
指定した価格に達したら売買する。
- 買い指値: 現在より安い価格で買いたいとき
- 売り指値: 現在より高い価格で売りたいとき

### 逆指値注文（Stop Order）
損切りや、ブレイクアウトのエントリーに使う。
- 買い逆指値: 現在より高い価格で買う（上昇ブレイク狙い）
- 売り逆指値: 現在より安い価格で売る（損切り）

### OCO注文（One Cancels the Other）
2つの注文を同時に出し、片方が約定したらもう片方がキャンセルされる。
利確と損切りを同時に設定するときに使う。

## 損益計算の基本

```
損益 = (決済価格 - エントリー価格) × ロットサイズ × 通貨単位
```

例: USD/JPYを150.00で1ロット（10万通貨）ロング → 150.50で決済
```
(150.50 - 150.00) × 100,000 = 50,000円の利益
```

例: USD/JPYを150.00で0.1ロット（1万通貨）ロング → 149.50で損切り
```
(149.50 - 150.00) × 10,000 = -5,000円の損失
```

## 重要な概念: 証拠金とロスカット

### 必要証拠金
ポジションを持つために必要な最低限の資金。

```
必要証拠金 = 取引額 ÷ レバレッジ
```

例: USD/JPY 150円で1万通貨、レバレッジ25倍
```
150 × 10,000 ÷ 25 = 60,000円
```

### 証拠金維持率
```
証拠金維持率 = 有効証拠金 ÷ 必要証拠金 × 100%
```

### ロスカット
証拠金維持率が一定水準（業者により50-100%）を下回ると、
ポジションが強制的に決済される。**これが最悪のシナリオ。**

→ **レバレッジを低く保ち、損切りを必ず設定することで防ぐ**

## このプロジェクトで使う通貨ペア

初期は以下に限定する:

1. **USD/JPY** — 最も馴染みがあり、スプレッドが狭い
2. **EUR/USD** — 世界最大の取引量、トレンドが出やすい

理由: メジャーペアはデータが豊富で、バックテストの信頼性が高い。
