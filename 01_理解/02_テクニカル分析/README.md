# テクニカル分析の基礎

## テクニカル分析とは

過去の価格データ（チャート）からパターンを見つけ、将来の値動きを予測する手法。
「価格はすべての情報を織り込んでいる」という前提に立つ。

## ローソク足（Candlestick）

最も基本的なチャート表現。1本が一定期間の値動きを示す。

```
   ┃  ← 上ヒゲ（高値）
 ┏━┛
 ┃    ← 実体（始値〜終値）
 ┗━┓
   ┃  ← 下ヒゲ（安値）
```

- **陽線**（白/緑）: 終値 > 始値（上昇）
- **陰線**（黒/赤）: 終値 < 始値（下落）

### 時間足
- 1分足、5分足、15分足 — スキャルピング/デイトレ向け
- 1時間足、4時間足 — スイングトレード向け（**我々の主戦場**）
- 日足、週足 — 長期トレンド確認用

## 移動平均線（Moving Average）

一定期間の価格の平均値を線で結んだもの。**最も重要な指標の一つ。**

### 単純移動平均（SMA: Simple Moving Average）
```
SMA(N) = 直近N本の終値の合計 ÷ N
```

### 指数加重移動平均（EMA: Exponential Moving Average）
直近の価格により大きな重みをつける。SMAより反応が早い。

### よく使われる期間
| 期間 | 用途 |
|------|------|
| 20 | 短期トレンド |
| 50 | 中期トレンド |
| 200 | 長期トレンド（**機関投資家が注目**） |

### ゴールデンクロス / デッドクロス
- **ゴールデンクロス**: 短期MAが長期MAを下から上に突き抜ける → 買いシグナル
- **デッドクロス**: 短期MAが長期MAを上から下に突き抜ける → 売りシグナル
- **注意**: 単独では勝率が低い。他の指標と組み合わせる。

## ボリンジャーバンド（Bollinger Bands）

移動平均線の上下に標準偏差（σ）の帯を描いたもの。

```
上バンド = SMA(20) + 2σ
中バンド = SMA(20)
下バンド = SMA(20) - 2σ
```

- 価格の約95%は±2σの範囲に収まる（統計的性質）
- **バンドの外に出る** → 異常な動き（平均回帰の可能性）
- **バンドが収縮（スクイーズ）** → 大きな動きの前触れ

## RSI（Relative Strength Index）

「売られすぎ」「買われすぎ」を数値化する指標。0-100の範囲。

```
RSI = 100 - (100 ÷ (1 + RS))
RS = N期間の平均上昇幅 ÷ N期間の平均下落幅
```

| RSI値 | 解釈 |
|-------|------|
| 70以上 | 買われすぎ → 下落の可能性 |
| 30以下 | 売られすぎ → 上昇の可能性 |

**注意**: 強いトレンド中はRSI 70以上が長期間続くことがある。逆張りは危険。

## MACD（Moving Average Convergence Divergence）

2つのEMAの差からトレンドの強さと方向を判断する。

```
MACDライン = EMA(12) - EMA(26)
シグナルライン = MACDラインのEMA(9)
ヒストグラム = MACDライン - シグナルライン
```

- MACDがシグナルを上抜け → 買いシグナル
- MACDがシグナルを下抜け → 売りシグナル
- **ダイバージェンス**: 価格は高値更新なのにMACDは更新しない → トレンド転換の予兆

## ATR（Average True Range）

ボラティリティ（値動きの激しさ）を測る指標。
**損切り幅の設定に使う。本プロジェクトで最重要。**

```
True Range = max(高値-安値, |高値-前日終値|, |安値-前日終値|)
ATR(N) = True RangeのN期間平均
```

使い方:
- 損切り = エントリー価格 ± ATR × 1.5〜2.0
- ATRが大きい → ボラ高い → ポジションサイズを小さくする

## サポート/レジスタンス

- **サポート（支持線）**: 価格が下げ止まりやすい水準
- **レジスタンス（抵抗線）**: 価格が上げ止まりやすい水準
- これらを突破すると、大きな動きにつながることが多い（ブレイクアウト）

## このプロジェクトで使う指標

| 指標 | 用途 |
|------|------|
| EMA(20, 50, 200) | トレンド方向の判断 |
| ATR(14) | 損切り幅・ポジションサイズの計算 |
| RSI(14) | エントリーフィルター（極端な過熱を避ける） |
| MACD(12,26,9) | トレンド転換の確認 |

**重要**: 指標は「補助ツール」であり、これだけで勝てるわけではない。
戦略全体のルール（エントリー・決済・リスク管理）が揃って初めて機能する。
