# リスク管理 — 最も重要な章

## なぜリスク管理が最重要か

どんなに優れた戦略でも、**勝率100%はあり得ない**。
プロのトレーダーでも勝率は40-60%程度。

つまり、負けることを前提に「負けても退場しない仕組み」を作ることが、
勝ち続けるための唯一の方法。

> 「市場で生き残ることが、市場で稼ぐための前提条件である」

## 3つの鉄則

### 鉄則1: 1トレードあたりのリスクは口座の1-2%以下

```
口座資金: 100万円
1トレードの最大損失: 1万〜2万円
```

これを守ると:
- 10連敗しても口座の80-90%が残る
- 20連敗しても口座の60-80%が残る
- 統計的に、10連敗は十分起こりうる

### 鉄則2: リスクリワード比は最低1:1.5以上

```
リスク（損切り幅）: 50pips
リワード（利確幅）: 75pips以上
```

リスクリワード1:2の場合、勝率34%でもトントン。
勝率50%なら大幅にプラスになる。

| 勝率 | RR 1:1 | RR 1:1.5 | RR 1:2 |
|------|--------|----------|--------|
| 40% | -20% | -10% | ±0% |
| 50% | ±0% | +25% | +50% |
| 60% | +20% | +50% | +80% |

（100トレード、1トレード1万円リスクの場合の損益）

### 鉄則3: 必ず損切りを設定する

- エントリーと同時に必ずストップロスを設定する
- **「もう少し待てば戻るかも」は破滅への道**
- 損切りを動かすのは利益方向のみ（トレーリングストップ）

## ポジションサイズの計算

**最も実践的で重要な計算。**

```
ポジションサイズ = (口座資金 × リスク率) ÷ (損切り幅pips × 1pipあたりの価値)
```

### 具体例

```
口座資金: 100万円
リスク率: 1%（= 1万円）
損切り幅: 50pips
通貨ペア: USD/JPY（1pip = 100円/1万通貨）

ポジションサイズ = 10,000 ÷ (50 × 100) = 2万通貨（0.2ロット）
```

もしATRが大きく損切り幅が100pipsなら:
```
ポジションサイズ = 10,000 ÷ (100 × 100) = 1万通貨（0.1ロット）
```

→ **ボラティリティが高いときは自動的にポジションが小さくなる**

## ドローダウン管理

### ドローダウンとは
口座の最高値からの下落率。

```
最高値: 120万円
現在値: 100万円
ドローダウン: (120 - 100) / 120 = 16.7%
```

### ルール
- **最大ドローダウン20%でトレード停止** → 戦略の見直し
- **最大ドローダウン30%で運用中止** → バックテストからやり直し

### ドローダウンからの回復に必要な利益率

| ドローダウン | 回復に必要な利益率 |
|------------|-------------------|
| 10% | 11.1% |
| 20% | 25.0% |
| 30% | 42.9% |
| 50% | 100.0% |

→ **ドローダウンが大きいほど回復が指数的に困難になる**

## 資金管理の実装方針

本プロジェクトでは以下をコードで強制する:

```python
# 疑似コード
class RiskManager:
    MAX_RISK_PER_TRADE = 0.01      # 口座の1%
    MAX_RISK_TOTAL = 0.05           # 全ポジション合計で5%
    MAX_DRAWDOWN_WARN = 0.15        # 15%で警告
    MAX_DRAWDOWN_STOP = 0.20        # 20%で停止
    MIN_RISK_REWARD = 1.5           # 最低リスクリワード比

    def calculate_position_size(self, balance, stop_loss_pips, pip_value):
        risk_amount = balance * self.MAX_RISK_PER_TRADE
        return risk_amount / (stop_loss_pips * pip_value)

    def can_open_trade(self, current_exposure, drawdown):
        if drawdown >= self.MAX_DRAWDOWN_STOP:
            return False  # トレード停止
        if current_exposure >= self.MAX_RISK_TOTAL:
            return False  # リスク上限到達
        return True
```

## メンタルリスク管理

自動売買でもメンタルは重要:

1. **システムを信頼する**: バックテストで検証済みなら、短期の負けで変更しない
2. **ルールを破らない**: 「今回だけロットを増やす」は破滅の入口
3. **記録をつける**: すべてのトレードを記録し、感情的な判断がなかったか振り返る
