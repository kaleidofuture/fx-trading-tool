# FAQ — よくある疑問と回答

学習過程で出てきた疑問を記録していく。

---

## Q1. 国内FX業者の最大レバレッジ25倍は、実績値？理論値？

**A. どちらでもなく、法規制による上限値。**

- **根拠**: 金融商品取引業等に関する内閣府令 第117条第1項第27号の2
- **経緯**:
  - 規制前（〜2010年）: レバレッジ100倍〜400倍の業者も存在していた
  - 2010年8月: 最大50倍に規制
  - 2011年8月: 最大**25倍**に引き下げ（現行）
- **対象**: 金融庁に登録されたすべての国内FX業者（個人口座）
- **法人口座**: 別基準。通貨ペアごとにリスクに応じて変動（概ね50〜100倍程度）

### なぜ規制されたのか

規制前は高レバレッジで多額の損失を出す個人投資家が続出し、社会問題化した。
金融庁が投資家保護の観点から段階的に引き下げた。

### 海外業者との違い

海外FX業者は日本の法規制の対象外で、レバレッジ500倍〜1000倍を提供する業者もある。
ただし以下のリスクがある:

- 金融庁の投資家保護の対象外
- 出金トラブルの報告が多い
- 信託保全が義務付けられていない

**本プロジェクトでは国内業者（25倍以内）を使用する方針。**
そもそも我々の戦略ではレバレッジ3〜5倍程度しか使わないため、25倍の上限は問題にならない。

## Q2. EUR/USD はユーロで買って米ドルで売るという意味？

**A. そうではない。EUR/USD は「1ユーロが何ドルか」を示す価格表示。**

通貨ペアの読み方:

```
EUR/USD = 1.0850
→「1ユーロ = 1.0850ドル」という意味
```

- **左側（EUR）= 基軸通貨**: 「何を」売買するか
- **右側（USD）= 決済通貨**: 「何で」値段を表すか

### 売買の意味

| 操作 | やっていること | 期待する値動き |
|------|--------------|---------------|
| EUR/USDを **ロング（買い）** | ユーロを買い、同時にドルを売る | EUR/USD の数字が上がる（ユーロ高・ドル安） |
| EUR/USDを **ショート（売り）** | ユーロを売り、同時にドルを買う | EUR/USD の数字が下がる（ユーロ安・ドル高） |

つまり通貨ペアの売買は**常に2つの通貨が同時に動く**。
「EUR/USDを買う」=「ユーロを買ってドルを売る」という1つの取引。

### USD/JPY の場合

```
USD/JPY = 150.00
→「1ドル = 150円」

ロング（買い）= ドルを買い、円を売る → 円安になれば利益
ショート（売り）= ドルを売り、円を買う → 円高になれば利益
```

### なぜこの順番なのか

通貨ペアの順番は国際的な慣習で決まっている。
EUR > GBP > AUD > NZD > USD > JPY の優先順位で、左に優先度が高い通貨が来る。

だから EUR/USD であって USD/EUR とは書かない。
USD/JPY であって JPY/USD とは書かない。

## Q3. 「ユーロを買い、同時にドルを売る」は本当に同時？タイムラグはある？

**A. 2つの意味を分けて理解する必要がある。**

### 「同時」の意味 — これは概念上の話

EUR/USDを「買う」というボタンを1回押すだけ。
内部的に「ユーロ買い＋ドル売り」が1つの取引として瞬時に成立する。
自分で2回の操作をする必要はない。タイムラグもない。

```
[EUR/USD 買い] ボタン1回 → 即座に約定
```

### 利益を得るまでの流れ — ここにはラグがある

FXで利益が出るのは「エントリー」と「決済」の2回のアクション。

```
① エントリー: EUR/USD を 1.0850 で買い（ロング）
   → ユーロを持ち、ドルを売った状態になる

   ～数時間、数日、数週間が経過～（ここがラグ）

② 決済: EUR/USD を 1.0900 で売り（クローズ）
   → ユーロを返し、ドルを受け取る
   → 差額 50pips が利益
```

つまり:
- **1回の取引（ユーロ買い＋ドル売り）の中** → 同時。ラグなし
- **エントリーと決済の間** → 意図的にラグを作る。ここで値動きを待つ

### 株との比較

```
株:   「トヨタ株を買う」→ 保有 → 「トヨタ株を売る」→ 利益確定
FX:   「EUR/USDを買う」→ 保有 → 「EUR/USDを売る」→ 利益確定
```

構造は同じ。「買って、しばらく持って、売る」。
ただしFXは「売りから入る（ショート）」こともできる点が株より柔軟。
