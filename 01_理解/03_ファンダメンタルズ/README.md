# ファンダメンタルズ分析

## ファンダメンタルズとは

経済指標・金融政策・地政学的イベントなど、通貨の「本質的な価値」に影響する要因。
テクニカル分析が「チャートの形」を見るのに対し、ファンダメンタルズは「なぜ動くか」を理解する。

## 通貨の価値を決める主要因

### 1. 金利（最重要）

通貨の価値に最も大きな影響を与えるのは**金利差**。

- 金利が高い国の通貨 → 資金が集まる → 通貨高
- 金利が低い国の通貨 → 資金が流出する → 通貨安

各国の政策金利を決める中央銀行:
| 国 | 中央銀行 | 略称 |
|----|---------|------|
| 米国 | 連邦準備制度理事会 | FRB/Fed |
| 日本 | 日本銀行 | BOJ |
| 欧州 | 欧州中央銀行 | ECB |
| 英国 | イングランド銀行 | BOE |

**重要なのは「現在の金利」ではなく「今後の金利の方向性」。**
市場は常に先を織り込む。

### 2. 経済指標

| 指標 | 影響度 | 内容 |
|------|--------|------|
| **雇用統計（NFP）** | ★★★ | 毎月第1金曜。米国の雇用者数。最大の市場イベント |
| **CPI（消費者物価指数）** | ★★★ | インフレ率。金利政策に直結 |
| **GDP** | ★★☆ | 経済成長率。四半期ごと |
| **PMI（購買担当者景気指数）** | ★★☆ | 50以上=景気拡大、50未満=縮小 |
| **中央銀行の声明/議事録** | ★★★ | 金融政策の方向性を示唆 |

### 3. 地政学リスク
- 戦争・紛争 → リスクオフ → 円・ドル・スイスフランが買われる
- 政治的不安定 → その国の通貨安

### 4. リスクオン / リスクオフ

**リスクオン**（市場が楽観的）:
- 高金利通貨（AUD, NZD）が買われる
- 円・スイスフランが売られる

**リスクオフ**（市場が悲観的）:
- 安全通貨（JPY, USD, CHF）が買われる
- 高金利通貨が売られる

## 経済指標カレンダーの読み方

経済指標は事前に発表日時がわかる。

```
日時: 2026-03-08 22:30 (JST)
指標: 米国 非農業部門雇用者数（NFP）
予想: +180K
前回: +151K
結果: ???
```

- **結果 > 予想** → ポジティブサプライズ → その国の通貨高
- **結果 < 予想** → ネガティブサプライズ → その国の通貨安
- **結果 ≒ 予想** → すでに織り込み済み → 反応薄

## 本プロジェクトでのファンダメンタルズの扱い方

我々のシステムは**テクニカルベースの自動売買**が主軸。
ファンダメンタルズは以下の形で組み込む:

### やること
1. **重要指標発表前後はトレードしない**（フィルター）
   - NFP、CPI、FOMC発表の前後30分はエントリーしない
   - 予測不可能な急変動を回避する
2. **金利差の方向をトレンドフォローの補助情報にする**
   - 金利上昇国の通貨をロングする方がトレンドに乗りやすい
3. **経済指標カレンダーの自動取得**
   - APIで指標スケジュールを取得し、フィルターに反映

### やらないこと
- 指標結果を予測してトレードする（→ ギャンブルになる）
- ニュースを読んでAIに判断させる（→ 機関投資家のほうが早い）
