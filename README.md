# FX × AI トレーディングツール（検証プロジェクト）

FXの自動売買は本当に機能するのか？「予測で勝つ」のではなく「統計的エッジを小さく積み重ねる」をコンセプトに、戦略設計からバックテスト、デモ運用まで一貫して検証するプロジェクトです。

> **注意**: これは学習・検証目的のプロジェクトです。投資助言や実際の取引を推奨するものではありません。

## 特徴

- **Dual Momentum Trend Following 戦略** — EMA 20/50/200、RSI、ATR を組み合わせたルールベース戦略
- **自作バックテストエンジン** — 外部ライブラリ非依存のイベント駆動型エンジン（円建て損益統一）
- **AI vs ルールベース比較検証** — Claude CLI を使ったAI判断との比較（結論: ルールベースの方が優秀）
- **自動デモトレーダー** — シグナル検出 → 自動建玉 → SL/TP/トレーリング → 自動決済
- **Streamlit ダッシュボード** — リアルタイム為替表示、シグナル履歴、パフォーマンス分析

## プロジェクト構成

```
01_理解/                — FXの基礎知識・戦略の学術的根拠
02_開発/
  01_データ取得/        — yfinance による過去データ取得
  02_バックテスト基盤/  — 自作イベント駆動型バックテストエンジン
  03_戦略実装/          — Dual Momentum Trend Following 戦略
  04_シグナル通知/      — 4H足シグナルモニター
  05_ダッシュボード/    — Streamlit + Plotly ダッシュボード
  06_AI検証/            — AI判断 vs ルールベースの比較検証
03_運用/

  02_デモトレーダー/    — 自動デモトレーディングエンジン
```

## 技術スタック

- Python 3.12
- pandas / numpy / yfinance
- streamlit / plotly
- matplotlib
- 自作バックテストエンジン（外部ライブラリ非依存）

## リスク管理

- 1トレードあたり口座の **1%** をリスク上限に設定
- 最大同時ポジション **3**
- ドローダウン **20%** で自動停止
- ATR ベースのトレーリングストップ

## セットアップ

```bash
pip install pandas numpy yfinance streamlit plotly matplotlib plyer
```

### ダッシュボード起動

```bash
cd 02_開発/05_ダッシュボード

```

### デモトレーダー実行

```bash
cd 03_運用/02_デモトレーダー
python demo_trader.py --once
```
