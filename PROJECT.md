# FX × AI トレーディングツール開発プロジェクト

## 方針
- 「予測で勝つ」のではなく「統計的エッジを小さく積み重ねる」
- すべての戦略はバックテストで検証してから実行する
- リスク管理が最優先。1トレードで口座の1%をリスクに晒さない

## プロジェクト構成

```
01_理解/              — FXの基礎知識・戦略の学術的根拠
02_開発/
  01_データ取得/      — yfinance による過去データ取得
  02_バックテスト基盤/ — 自作イベント駆動型バックテストエンジン
  03_戦略実装/        — Dual Momentum Trend Following 戦略
  04_シグナル通知/    — 4H足シグナルモニター（通知+ログ）
  05_ダッシュボード/  — Streamlit + Plotly ダッシュボード
  06_AI検証/          — AI判断 vs ルールベースの比較検証
  reports/            — バックテストレポート画像
03_運用/
  01_デモトレード記録/ — OANDA手動トレード記録（trade_log.py）
  02_デモトレーダー/   — 自動デモトレーディングエンジン
```
