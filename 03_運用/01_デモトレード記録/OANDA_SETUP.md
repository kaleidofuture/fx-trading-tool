# OANDA Japan デモ口座開設ガイド

## 手順

### 1. デモ口座を開設する

1. https://www.oanda.jp/trade/web/openAccount/mailAddressCheck?type=Demo にアクセス
2. メールアドレスを入力 →「メールを送信する」
3. 届いたメールのリンクをクリック
4. 申込画面で必要情報を入力して完了

**注意: デモ口座は開設後30日間有効**

### 2. NYサーバーのサブアカウントを作成

マイページにログイン後:
1. サブアカウント一覧を確認
2. **NYサーバー** のサブアカウントを作成する
   - APIトークン発行にはNYサーバーのアカウントが必要

### 3. APIトークンを発行する

1. マイページで「**APIアクセスの管理**」をクリック
2. API契約書に同意
3. 「**トークンを発行する**」をクリック
4. 表示されたトークンを**即座にコピーして保存**
   - 画面を離れると二度と表示されない（再発行は可能）

### 4. config.json に設定する

```json
{
    "oanda_token": "ここにトークンを貼り付け",
    "oanda_account_id": "xxx-xxx-xxxxxxx-xxx",
    "environment": "practice"
}
```

`oanda_account_id` はマイページのアカウント情報に表示される。

### 5. 接続テスト

```bash
cd 02_開発/04_シグナル通知
python oanda_client.py --test
```

## 重要な注意事項

- デモ口座の資金は仮想（リアルマネーではない）
- 本番口座への切り替えは Phase 3 で最低3ヶ月のデモ検証後
- config.json は .gitignore に追加済み（トークンを公開しないこと）
